# Grocery Chain Sales Data Analysis

Performed comprehensive Exploratory Data Analysis (EDA) on grocery chain sales data to uncover revenue trends, product performance, and discount effectiveness.

### Features

* Data cleaning and preprocessing using Pandas.
* Revenue analysis by store and monthly trends.
* Discount and product profitability analysis.
* Weekday vs. weekend and time-based sales analysis.
* Generated 12 interactive Plotly visualizations (bar, line, pie, scatter charts).

### Technologies Used

**Python, Pandas, NumPy, Plotly**

### Output

* Key metrics and insights in console.
* Interactive HTML charts for visual exploration.
